# Security Policy

## Project Status

This is a custom fork of [usememos/memos](https://github.com/usememos/memos), currently in active development.

## Reporting Security Issues

Please report security vulnerabilities via [GitHub Issues](https://github.com/albrtbc/memos/issues) with the **security** label.

**Include in your report:**
- Description of the issue
- Steps to reproduce
- Affected versions
- Your assessment of severity

## Self-Hosting Security

Since Memos is self-hosted software:
- Keep your instance updated to the latest release
- Don't expose your instance directly to the internet without authentication
- Use reverse proxies (nginx, Caddy) with rate limiting
